# svlang

Swedish NLP toolkit for translators. Spell checking, grammar, declension.

Built with Python CLI. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install svlang
```

### Fedora/RPM
```bash
sudo dnf install svlang
```
